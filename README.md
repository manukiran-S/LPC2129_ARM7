# LPC2129 ARM7 Projects

A collection of Embedded C projects developed on the LPC2129 (ARM7) microcontroller using Keil µVision 5 and LPC2129 Development Board.

## > Tools Used

* Keil µVision 5
* Embedded C
* LPC2129 Development Board
* Proteus 8 (for simulation)

## > Topics Covered

* LED Interfacing
* Switch Interfacing
* LCD Interfacing
* UART Communication
* ADC (Analog-to-Digital Converter)
* External Interrupts
* Timers
* GPIO Programming

## > Mini Projects Included

This repository contains multiple hands-on projects demonstrating the fundamentals of embedded systems programming and peripheral interfacing on the LPC2129 ARM7 microcontroller.

## > Learning Outcomes

* ARM7 Architecture Basics
* Peripheral Programming
* Interrupt Handling
* Serial Communication
* Embedded C Development
* Hardware-Software Integration
